
# Sellapy Global Technology (SGT) Limited

A Birmingham-based product and AI engineering website built with Next.js 14 App Router, TypeScript, and Tailwind CSS.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Visit the site**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app`: App Router pages and API routes.
- `/components`: Reusable UI components.
- `/lib`: Helper utilities and business logic.
- `/data`: JSON-based storage for local contact form submissions.

## Design System

- **Clean Typography**: Using system fonts for a professional, high-end feel.
- **Color Palette**: Near-black (#020617) accents on off-white backgrounds with subtle Indigo gradients.
- **Responsiveness**: Mobile-first design using Tailwind's grid and flexbox.
